Silverline

Integrating Ruby on Rails and Silverlight

Using the plugin

Currently the plugin ships with a sample application wrapped around it (demos below). To use the plugin on your own, just copy the vendor/plugins/silverline directory to your own application.

Note: you will need two things installed:

  • FileSystemWatcher

  • gem install rubyzip | sourceforge
